# article-summarizer
## Overview

AI Article Summarizer is a web application that uses the Google Gemini API to summarize long articles. Users can paste an article and instantly receive a TLDR summary, bullet points, key facts, and sentiment analysis in an easy-to-read format.

---

## Features

* Generate AI-powered article summaries
* Display one-line TLDR summaries
* Extract 3–5 bullet points
* Show important key facts
* Perform sentiment analysis (Positive, Negative, Neutral)
* Display article word count
* Copy generated summary to clipboard
* Loading and error handling

---

## API Used

### Google Gemini API

The application uses the **Google Gemini 2.5 Flash** model to generate article summaries and analyze content.

**Features of Gemini API:**

* Free API key available
* Fast response time
* High-quality summaries
* Easy integration with JavaScript
